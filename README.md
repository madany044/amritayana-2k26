# 🌌 AMRITAYANA-2K26

> A premium, highly interactive dark-themed web portal built to power the annual college fest event registration and live standings. Crafted with **React, Vite, Framer Motion, and Firebase Firestore**.

🌌 **[Launch Live Portal](https://amritayan-2k26.vercel.app/)**

---

### 🎉 Proven in Production
This application served as the central hub for the college fest, seamlessly registering **390+ students** and serving real-time updates with absolute speed and reliability.

---

## ✨ Key Highlights & Architectural Features

### 🏆 Real-Time Live Leaderboard
- **Dynamic Standings**: Automatically aggregates results from Firebase Firestore collections to compute and display department/branch-wise standings.
- **Instant Synchronization**: Any updates on event outcomes immediately reflect on the public dashboard, keeping the entire campus engaged in real time.

### 📋 Adaptive Registration Engine
- **Morphing Forms**: A smart form interface that dynamically restructures itself based on event metadata (Solo, Duet, or Group/Team events).
- **Intelligent Validation**: Collects team details, handles dynamic player inputs, and ensures data integrity before dispatching to the cloud database.

### 🔍 Fluid Event Discovery Hub
- **Seamless Filtering**: Features a high-performance search and category filtering system (Solo / Doubles / Team) for frictionless exploration.
- **Modern Dark Aesthetic**: Implements polished glassmorphic components, horizontal-scrolling cards with background overlays, and fluid micro-animations.

### 🛠️ Robust Database & Operations
- **Real-Time NoSQL Architecture**: Built on Firestore to manage concurrent requests smoothly under high campus traffic.
- **Operational Scripts**: Equipped with custom database management utilities (`firebase_seed.js`, `update_registrations.js`, and `clear_results.js`) to support admin operations seamlessly.

---

## 🎨 Tech Stack & Design System

- **Frontend Core:** React 18 & Vite (Ultra-fast bundler for modern web performance)
- **Styling Paradigm:** Pure Vanilla CSS with CSS Modules (Ensuring modular scoped styling, pixel-perfect responsiveness, and zero layout shift)
- **Fluid Motions:** Framer Motion 
- **Cloud Backend:** Firebase Firestore 
- **Deployment:** Vercel
